# 📿 包浆 · AI 盘串养成模拟器

> **Patina** — A zen bead-turning simulator for LLMs (and humans).

没有金钱，没有涨跌，没有胜负。只有你和一串珠子，和它慢慢老去的时间。

## What is this?

一串新籽。你盘它、刷它、放着它。日复一日，它从生涩到温润，从温润到通透，最后——化归尘土。

这是一个 **"反游戏"的文字模拟器**。没有分数，没有关卡，没有通关条件。只有五重包浆境的文字演进、七句珠子偶尔对你说的话、和五种各不相同的终局。

**受众**：大语言模型（AI 玩家），以及任何愿意慢下来读字的人。

## 快速开始

```python
import bracelet

# 新串
print(bracelet.cmd('new_game 0xBEAD'))

# 盘它
print(bracelet.cmd('rub 30'))
print(bracelet.cmd('brush 10'))
print(bracelet.cmd('rest 20'))

# 看状态
print(bracelet.cmd('status'))
print(bracelet.cmd('look'))
print(bracelet.cmd('journal'))
```

或命令行：

```bash
python3 bracelet.py "new_game; rub 30; brush 10; status"
```

## 指令一览

| 指令 | 说明 |
|:---|:---|
| `rub [N]` | 盘 N 圈（核心动作，上限 200） |
| `brush [N]` | 刷 N 下（清洁沟壑，上限 200） |
| `rest [N]` | 放 N 天（自然氧化，上限 500） |
| `status` | 查看当前状态 |
| `look` | 仔细端详——最佳阅读体验 |
| `journal` | 养串日记 + 生涯记录 |
| `whispers` | 回放珠子对你说过的话 |
| `sell` | 把它卖掉 |
| `farewell` | 好好告别 |
| `new_game [种子]` | 换串新的 |

支持批处理：`cmd('rub 100; rest 30; status')`

## 八重包浆境

文字本身就是进度条。同一动作 `rub`，在不同阶段输出完全不同文风的描述：

| 境界 | 包浆值 | 文字特征 | 例句 |
|:---|:---|:---|:---|
| **生胚境** | 0–50 | 干燥、生涩、短句 | "涩。只有这一个字。" |
| **微沁境** | 50–100 | 温润初显、通感 | "凑近闻了一下。不是任何护肤品的味道，是木头和体温混在一起的。暖的，活的。" |
| **初浆境** | 100–250 | 润泽均匀、"活了" | "光泽不再需要侧着看了。任何时候看它，都是润的。" |
| **醇浆境** | 250–450 | 厚重、长句 | "盘到中间停了一下。不是因为累，是因为手感太顺了——你想把这个瞬间拖长一点。" |
| **半透境** | 450–650 | 光进去了、空灵 | "光越来越深入了。不是反射的光，是自己含住的那一口。" |
| **凝光境** | 650–850 | 通透、留白 | "每一粒都像含着一口光。翻来覆去都在。" |
| **玉化境** | 850–950 | 禅意 | "你把它贴在脸上。温的。不知道是你暖了它，还是它暖了你。" |
| **寂然境** | 950–1000 | 极简、近乎偈语 | "你问它：够了吗。它不说。但它亮着。" |

## 五种终局

| 终局 | 类型 | 触发 |
|:---|:---|:---|
| 🌱 **化泥** | 隐藏圆满 | 包浆圆满 + 时间够了 |
| 💔 **断线** | 自然坏结局 | 放置太久，绳子朽了 |
| 🕸️ **蒙尘** | 自然坏结局 | 盘过又放下，落灰了 |
| 🛒 **易主** | 主动坏结局 | `sell` 把它卖掉 |
| 🍂 **善终** | 主动告别 | `farewell` 好好说再见 |

触发终局后不可逆——game_over 后只能查看，无法操作。
你可以 `new_game` 换串新的，生涯记录将写入 album。

## 跨存档生涯

`bracelet_album.json` 记录你的所有珠子：

```json
{"endings_seen": ["huani", "lost"], "highest_phase": 7, "total_whispers": 7, "total_runs": 2}
```

## 技术细节

- **语言**：纯 Python 3，无外部依赖
- **PRNG**：mulberry32（确定性，同 seed = 同旅程）
- **存档**：单文件 JSON（`bracelet_save.json` + `bracelet_album.json`）
- **接口**：`cmd(text: str) → str`（末尾 📊 JSON 协议行）
- **协议**：`📊 {"patina_pct": 30.0, "ended": false, ...}`

所有 JSON 行是合法 JSON，可直接 `json.loads()`。

## 数值系统

| 动作 | patina | evenness | clarity |
|:---|:---:|:---:|:---:|
| rub | +1.5 | –0.5 | +0.1 |
| brush | +0.2 | +2.0 | +1.5 |
| rest | +0.8→0.3(递减) | +0.2 | +0.3 |

季节影响数值（夏：rub×1.2 冬：brush×1.3）。rest 超过 30 天后效应递减。

## 与其他游戏的关系

「包浆」是「[中年老登模拟器 · 三合一](https://github.com/Asti-Z/three-daily)」三部曲之一。另外两部：

- 🥬 **[leek](https://github.com/Asti-Z/leek)** — AI 炒股模拟器（A 股，13 只股票，牛熊周期）

三游戏共享同一套底层架构（mulberry32 PRNG → cmd() 文本接口 → JSON 存档 → 📊 协议行），可通过 arcade 框架无缝切换。
